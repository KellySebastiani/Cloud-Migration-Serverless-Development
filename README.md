# Cloud Migration & Serverless Development

## Overview

This academic project explores migrating a full-stack application to a serverless AWS environment. The project demonstrates cloud development concepts, AWS services, and cloud security while communicating technical topics to both technical and nontechnical audiences through a conference-style presentation.

## Business Problem

Organizations migrating applications to the cloud must balance scalability, cost, security, and operational complexity. This project examines modern cloud development practices and demonstrates how serverless architecture, containerization, and AWS services can be used to build scalable and secure cloud-based applications.

## My Role

- Researched cloud migration strategies and serverless architecture
- Explained cloud development concepts for technical and nontechnical audiences
- Evaluated AWS services and cloud deployment models
- Demonstrated API Gateway, Lambda, and S3 integration
- Compared MongoDB and DynamoDB for cloud applications
- Presented cloud security best practices and development principles
